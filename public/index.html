nk-book

A 3-layer AI pipeline that takes scanned Korean book pages and produces verified English translations. OCR, translation with voice preservation, and multi-agent validation.

View on GitHub

How it works

1

OCR Extraction

Sends each scanned page photo to Claude's vision API. Extracts Korean text exactly as written, preserving paragraph breaks and formatting. Handles HEIC, JPEG, and PNG. Converts PDFs to page images at 300 DPI.

2

Translation

Splits text into 3-page chunks and translates with sliding-window context for continuity. Maintains a glossary of proper nouns and recurring terms across the entire book. Preserves the author's voice, not just words.

3

Validation

3 independent sub-agents check each chunk. Back-translation catches semantic drift. Tone scoring evaluates formality, emotion, voice, rhythm, and cultural fidelity. An editorial agent reconciles and grades. Chunks below threshold get auto-retranslated.

Features

Resumable

Every step saves progress. Crash mid-run, re-run, and it picks up where it left off.

Voice preservation

Translation prioritizes the author's voice over literal accuracy. Simple stays simple.

Auto-retranslation

Chunks scoring below 3.5 or with high-severity issues get re-translated automatically (max 2 attempts).

Side-by-side output

Generates Korean and English paragraphs interleaved for easy review.

Usage

# Extract pages from PDF python extract_pages.py # Run the full pipeline python pipeline.py --step all # Or run individual steps python pipeline.py --step ocr python pipeline.py --step translate python pipeline.py --step validate python pipeline.py --step assemble
